🏦

Dedicated Virtual Accounts

Each customer gets their own bank account number. Payments are automatically attributed — no manual matching.

⚙️

Automatic Reconciliation

Webhooks are confirmed against the provider in real time. Discrepancies are flagged, not buried.

📨

Reliable Webhooks

HMAC-signed, retrying, with delivery history. You get notified of every payment — exactly once.

💰

Wallets & Payouts

Track balances with a double-entry ledger. Pay out to any Nigerian bank account via NIP transfer.

🔒

API-Key Auth

Server-to-server calls use a single API key. No OAuth dance — just a Bearer token in the header.

📊

Developer Dashboard

Manage customers, view transactions, configure webhooks, and track reconciliation — all in one place.
